Margins

Margins can be set as just "margin" using px measurements, which changes the margin evenly around the object. Other settings included margin-top, margin-bottom, margin-left, and margin-right. You can use "auto" instead of px measurments to have an object stick to your specified side of the page no matter how big it is. So "margin-left: auto;" will cause the object to stick to the right side of the screen. Just "margin: auto;" will have the object in the middle of the screen.
